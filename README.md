# Triangulitation
A Triangulitation algorithm

# How to compile
$ make all

# How to execute
There are to mode:

$ ./triangulitation.o 0

that generate some random lines and *try* to triangulate it. and:

$ ./triangulitation.o 1

that will ask you some coordiante unitl you type 501 501
